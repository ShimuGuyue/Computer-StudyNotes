| 权限                 | 说明               |
| -------------------- | ------------------ |
| $ALL,ALL PRIVILEGES$ | 所有权限           |
| $SELECT$             | 查询数据           |

| $UPDATE$             | 修改数据           |
| $DELETE$             | 删除数据           |
| $ALTER$              | 修改表             |
| $DROP$               | 删除数据库/表/视图 |
| $CREATE$             | 创建数据库/表/视图 |
